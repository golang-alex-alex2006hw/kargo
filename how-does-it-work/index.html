    # How does it work ?

    kargo consists in a docker based (Compose or Swarm) solution which allows you to set up and deploy the following geospatial web applications:

    1. PostGis: https://hub.docker.com/r/mdillon/postgis/
    2. GeoServer: Kargo provides its own Dockerfile to build GeoServer
    3. MapProxy: https://hub.docker.com/r/yagajs/mapproxy/
    4. TileServer-GL: https://hub.docker.com/r/klokantech/tileserver-gl/
